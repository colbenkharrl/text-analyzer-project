Text Analyzer Pro:

Analyze your files and track statistics between different file formats
on your file system.  

From the main menu you can select 'Analyze' and you will be 
prompted to select a file to be analyzed.

Upon selection the files statistics will be reproted on screen, 
and this information will be recorded in the history tab
accesible through the main menu.
